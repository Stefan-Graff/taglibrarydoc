Release 1.3 – 04-12-2005

Type Changes By
Support for JSP 2.1 TLD schema. Support for new JSP 2.1 TLD elements, including: deferred value attributes and associated type, deferred method attributes, and associated signature. Fixes Issue 12. markroth8

Release 1.2 – 10-11-2004

Type Changes By
Now supports CDATA-escaped HTML tags in any JSP 1.1 info or JSP 1.2 / JSP 2.0 description tag. Note: If you have any customized stylesheets you will have to regenerate them for this release. For the most part, you just have to add the j2ee: prefix when referring to the input source. See the default stylesheets for details. Fixes Issue 2. markroth8
Fixed a bug that prevented taglibrarydoc from working with JDK 5.0. Fixes Issue 4. markroth8
