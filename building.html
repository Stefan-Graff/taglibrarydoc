Build Instructions

To build Tag Library Documentation Generator, follow these steps:

  1. Download and install maven 3.2.5 or greater from https://maven.apache.org/.
  2. Check out the taglibrarydoc source code from github.com. You can do so here: https://github.com/ste-gr/tlddoc
  3. Run ‘mvn clean package’. A target directory will be created with tlddoc-1.4-SNAPSHOT.jar.
  4. See the following instructions for how to use the Tag Library Documentation Generator.
